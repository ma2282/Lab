Команды Git
Команда Описание
git init Create an empty Git repository or reinitialize an existing one
git clone url Clone a repository into a new directory
git add file Add file contents to the index
git add . Add all file contents to the index
git commit -m "comment" Record changes to the repository with "comment" message
git commit -a -m "comment" Record changes to the repository with "comment" message, but new files you have not told Git about are not affected
git push url Update remote refs along with associated objects
git status Show the working tree status
git log Show commit logs
git branch branch_name List, create, or delete branch
git checkout branch_name Switch branches
git checkout -b branch_name Create new branch with branch_name
git remote -
